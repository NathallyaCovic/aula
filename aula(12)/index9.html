❓ Perguntas Frequentes

O que é HTML?

HTML é a linguagem usada para estruturar o conteúdo da web.

O que é CSS?

CSS serve para estilizar o HTML, controlando a aparência e o layout.

Posso usar HTML no celular?

Sim! Você pode criar e editar arquivos HTML em qualquer editor de texto.

O que é uma tag?

Uma tag é um elemento usado para marcar partes do conteúdo HTML.

HTML precisa de internet?

Não! Ele pode ser usado e testado offline no seu computador.
